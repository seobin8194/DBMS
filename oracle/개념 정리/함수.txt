단일행 함수
한 행(데이터)마다 결과가 하나씩 나오는 함수
- NULL처리함수
  NVL(컬럼, 값) : 컬럼이 NULL이면 뒤에 오는 값으로 표현 그 외에는 그대로 출력
  NVL2(컬럼, 값1, 값2) : 컬럼이 NULL인 경우 값2 그 외에는 값1 
  COALESCE(컬럼, 값) : NVL과 동일

- 문자열 함수
   CONCAT(문자열,문자열) : 문자열 연결. 두개씩만 연결할 수 있다
   또 다른 연결 (concatenation) ||
   : 컬럼이나 문자열을 연결할때 사용
    컬럼1 || 컬럼2 ->컬럼1컬럼2    문자열1 || 문자열2 -> 문자열1문자열2
   
   UPPER : 대문자로 LOWER : 소문자로 INITCAP : 첫 글자만 대문자로
   오라클에서는 대소문자 구분해서 비교하기때문에 사용자가 입력한 값과 DB에 입력한 값의 대소문자를 일치시킬때 사용

   LENGTH : 문자개수 LENGTHB : 문지열의 바이트 개수

   SUBSTR
   : 문자열 나누기
     SUBSTR(값, 시작 위치, 글자수)
     (인덱스는 1부터) 
   
   INSTRING
   : 문자열 찾기, 안에 문자열이 있는지
     INSTRING(문자열, 찾는 문자열, 탐색 시작 위치, 몇번째 등장) 
     -> 찾으면 그 문자열의 인덱스 반환(없으면 0) -> 없으면 0을 반환하므로 오라클의 인덱스는 1부터
   
   LPAD RPAD
   LPAD(문자열, 총 글자수, 채울 문자열) : 총 글자수에서 문자열빼고 남은 글자수를 왼쪽부터 채울 문자열로 채운다 
   RPAD(문자열, 총 글자수, 채울 문자열) : 총 글자수에서 문자열빼고 남은 글자수를 오른쪽부터 채울 문자열로 채운다

   REPLACE
   : 문자열 바꾸기  
     REPLACE(문자열, 바꿀 문자열, 새로운 문자열)
 
- 숫자 함수
   ABS() : 절대값
   SIGN()
   MOD() : 나머지 
   CEIL() : 값보다 큰 최근접 정수 (올림) 
   FLOOR() : 값보다 작은 최근접 정수 (내림)
   ROUND() : 반올림
   TRUNC() : 버림
   POWER() : 제곱

- 날짜 함수
  날짜 데이터도 기본 연산이 가능하다 (+1 반올림 등,,)
  MONTHS_BETWEEN() : 개월 차이
  LAST_DAY(SYSDATE) : 이번 달 마지막 날

- 형변환 함수
   TO_DATE('OOOO-OO-OO', '날짜서식') : 날짜를 서식에 맞는 DATE 타입의 데이터로 변경해준다
   TO_CHAR(DATE 타입 데이터, '날짜서식') : DATE 타입의 데이터를 날짜 형식에 맞는 문자열로 변경해준다

그룹함수
여러 데이터들을 받아서 하나의 결과를 도출하는 함수
   COUNT() : NULL을 제외한 개수를 센다
   AVG() : 평균 평균구할때 NULL이 포함되어 있으면 그냥 NULL이 나오므로 NULL처리를 해야한다 AVG(NVL(컬럼,0))
   MAX() : 최대값 
   MIN() : 최소값 
   SUM() : 총합
